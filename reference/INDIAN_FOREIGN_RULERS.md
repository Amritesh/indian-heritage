# Indian Foreign Rulers

## Summary

Reference note for the `indian-foreign-rulers` collection. The source scan set is organized as title/preface pages followed by mounted coin-card reference sheets for Portuguese India, Danish India, Dutch India, and Indo-French issues.

## Ingest Notes

- Use the handwritten card label as the primary metadata signal.
- Treat each page as a multi-card reference sheet unless the scan clearly shows a paired sequence that belongs together.
- Prefer historical ruler and mint labels over modern country labels when normalizing.
- Preserve the image as the media reference; do not infer a separate obverse/reverse pair unless the page makes that explicit.
